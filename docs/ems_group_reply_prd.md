# EMS 组命令回复与0x08聚合 PRD

**目标**
实现同组模块对组命令的回复，并在组命令读取 `0x08` 时返回聚合电流值，满足 EMS 读写需求。

**范围**
涉及模拟器协议层与数值回包逻辑，保持现有 CAN 报文格式不变。

**需求说明**
1. 组命令（设备号 `0x0B`、目标地址为组号）时，同组内所有模块都应回复。
2. 广播命令（目标地址 `0x3F`）不回复。
3. `0x08` 在组命令场景下返回系统电压与聚合电流值；电压取模块电压，电流取组内总和。
4. 单模块场景保持现有行为。

**数据与参数**
1. 新增参数 `group_size`，用于估算聚合电流。
2. `0x1B` 设定总电流后，模块电流按 `group_size` 平均分配。
3. `0x1C` 设定模块电流后，聚合电流为模块电流乘以 `group_size`。

**非目标**
1. 不引入真实负载模型或动态电流分配策略。
2. 不改动 EMS 侧协议实现。
