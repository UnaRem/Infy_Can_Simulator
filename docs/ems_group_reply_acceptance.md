# EMS 组命令回复与0x08聚合 验收标准

**功能验收**
1. 组命令（设备号 `0x0B`、目标地址为组号）发送后，同组内所有模块均回包。
2. 广播命令（目标地址 `0x3F`）无回包。
3. 读取 `0x08` 时：
4. 点对点返回模块自身电压/电流。
5. 组命令返回模块电压与聚合电流值。

**边界验收**
1. `group_size=1` 时，`0x08` 聚合电流与模块电流一致。
2. 发送 `0x1B` 设定总电流后，`0x08` 聚合电流与设定值一致（允许微小浮点误差）。
3. 发送 `0x1C` 设定模块电流后，`0x08` 聚合电流为模块电流乘以 `group_size`。
